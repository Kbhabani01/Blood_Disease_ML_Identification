📖 About Project

This project uses Machine Learning to analyze blood parameters like RBC, WBC, hemoglobin, and platelets to detect diseases such as anemia and leukemia.

⚙️ Features

  • Disease Prediction using ML
  • Multiple Algorithms (Random Forest, SVM)
  • Data Visualization
  • Clinical Decision Support

🧠 How It Works

Input blood data → ML model processes → predicts disease → gives recommendations.

📊 Results

Achieved ~92% accuracy using Random Forest model.

⚠️ Disclaimer

This is a Clinical Decision Support System and not a replacement for professional medical advice.
